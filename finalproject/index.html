Featured Lab: Responsive Layout

Lab 3 focused on adding basic CSS styling to a previously created blog page. I used external stylesheets to change the appearance of elements like headings, paragraphs, and links. This lab helped me to understand how CSS separates design from structure and how small styling changes can improve a page’s readability and appearance.
