Reflection on the Final Project

This final project helped me realize how much I have improved in web development. While I did not need to change any of my original labs, putting them all together into one organized and styled website still took time and effort. I focused on keeping the design consistent across pages and making sure the layout worked well on different screen sizes. To test the layout, I used the Inspect tool in Chrome and also viewed the site on my phone to make sure the content adapted to the screen size of the device.

Before this class, my only web experience was using Google Sites in a previous course. Building a site completely from scratch was a challenge, but it was also more rewarding because it gave me full control over the structure and design. Writing my own HTML and CSS helped me understand how real websites are built. I now feel more confident in my ability to create and troubleshoot my websites.

When I compare my first lab to my final lab, I can see the progress that I made. My code is more organized, the layouts are cleaner, and I have learned to use proper tags and accessibility practices. The visual structure of my pages has improved, and the content is easier to navigate.
