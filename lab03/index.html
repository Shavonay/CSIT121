My HTML Journey - Labs 1, 2 & 3

Published on February 19, 2025 by Shavonay Morgan

Who knew that building a website could be so easy? Even though this is a simple website with little customization, I learned a lot about the basics of HTML and the structure of websites. I also learned that semantic elements such as "article," "section," and "main" are a great way to improve the readability and organization of a webpage. Additionally, the resources on W3Schools are extremely helpful, easy to follow, and are a great addition to the module notes and content. Overall, this assignment and the practice exercises made me more excited to learn more about HTML.

Since my last publication, I have learned about CSS and its importance in web development. I learned that CSS was created to solve the problem created after tags such as "font" were added to the HTML 3.2 specification. I also learned that CSS greatly simplifies the process of styling web pages with the use of external stylesheet files. In addition to the external stylesheet, CSS can be added to HTML using inline and embedded styles within the (style) tag. Overall, I liked exploring CSS and I hope to learn more.

What I Learned

  1. The importance of opening and closing tags
  2. Visual Studio Code shortcuts such as "HTML:5"
  3. How to create lists using "ul," "ol," and "li"
  4. Why CSS was created
  5. The three different ways to apply CSS (Inline, Embedded, External)
  6. The different selectors (HTML, ID, Class)
