Lab 03

Published on February 19, 2025 by Shavonay Morgan

Since my last publication, I have learned about CSS and its importance in web development. I learned that CSS was created to solve the problem created after tags such as "font" were added to the HTML 3.2 specification. I also learned that CSS greatly simplifies the process of styling web pages with the use of external stylesheet files. In addition to the external stylesheet, CSS can be added to HTML using inline and embedded styles within the (style) tag. Overall, I liked exploring CSS and I hope to learn more.

What I Learned

  1. Why CSS was created
  2. The three different ways to apply CSS (Inline, Embedded, External)
  3. The different selectors (HTML, ID, Class)

Great resources for learning about CSS are listed below.

  • CSS reference from W3 Schools
  • Getting Started with CSS (MDN Web Docs)
